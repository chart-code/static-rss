# static-rss

https://roadtolarissa.com/static-rss/

## Dev

- Replace `subs.xml` with your own [OPML file](https://blog.feedly.com/opml/).
- `yarn`
- `bin/download.js && bin/parse.js` update feeds
- `npx hot-server` start a local server
